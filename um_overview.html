Overview (getting started)

This section will teach you what components a working skarphed environment consists of and how to set it up.

Components

A basic skarphed installation consists of a Firebird-Database that will be used to store the persistent data of your skarphed-managed websites or webapplications, a skarphed-repo which is a repository that provides you with modular extensions (modules) and website-designs (templates) and finally the skarphed-core itself, which is the application that runs on the server and answers your audience's HTTP-request.
